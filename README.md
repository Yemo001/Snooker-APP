# Snooker-APP
**Description**
A 2 player snooker game that runs in your browser with a scoring system whenever a ball is potted and some funcitons where you can reset the game and score by pressing '1' and move the red balls randomly by pressing '2' and random spawn all balls by pressing '3'. Every collision between the balls are detected in the console of the web browser.

**How to RUN**
Do Clone the whole repository and execute it in the VScode using a live server extension or use a p5 live server on web to try out.

**Tools**
This is the snooker app I make using p5.js libraries and basic html in VScode.
