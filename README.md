# Snooker-APP

## Description
A 2-player snooker game that runs in your browser with a scoring system whenever a ball is potted and some functions where you can reset the game and score by pressing ‘1’ and move the red balls randomly by pressing ‘2’ and random spawn all balls by pressing ‘3’. Every collision between the balls is detected in the console of the web browser.

## How to Run
Clone the whole repository and run it in VSCode using a Live Server extension, or use the p5 live server on web to try it out.

## Tools
Built using p5.js libraries and basic HTML in VSCode.
