# Snooker-APP

---

## Description

A 2-player snooker game that runs in your browser with a scoring system whenever a ball is potted.  
The game includes several functions:

- Press **`1`** → Reset the game and score  
- Press **`2`** → Randomly move the red balls  
- Press **`3`** → Randomly spawn all balls again  

Every collision between the balls is detected and logged in the browser console.

---

## How to Run

Clone the entire repository and run it in **VSCode** using a **Live Server** extension,  
or use the **p5.js live server** on the web to run and test the game.

---

## Tools

Built using:

- **p5.js libraries**  
- **HTML** inside VSCode
